## Chat que funciona como CV
### Se le puede hacer cualquier pregunta y dara respuesta sobre la experiencia laboral. En caso que el usuario especifique quien envia el chat o desde que empresa se quiere contactar, se envia un aviso a una aplicacion de celular sobre quien lo contacto.

## Descripcion:
Prueba de concepto usando diferentes agentes. Los mismos redactaran diferentes formas de mail y un agente administrador decidira cual es el apropiado. Habra un "servidor" monitoreando si existe alguna respuesta de los destinatarios.
## Pasos:
### 1. Ejecutar ngrok http 5000 en un CMD
### 2. Ejecutar server
### 3. Ejecutar server_iniciador.py en otro CMD
